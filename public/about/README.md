# About Section Background Image

## Cách sử dụng
Chỉ cần đặt ảnh vào folder này với tên: **about-bg.jpg** (hoặc .png, .webp)

## Yêu cầu
- Tên file: `about-bg.jpg` (hoặc `about-bg.png`, `about-bg.webp`)
- Kích thước khuyến nghị: 1920x800px hoặc lớn hơn
- Định dạng: JPG, PNG, hoặc WebP

## Vị trí
Ảnh này sẽ được sử dụng làm background cho section Giới thiệu
