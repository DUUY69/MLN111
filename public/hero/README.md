# Hero Background Image

## Cách sử dụng
Chỉ cần đặt ảnh vào folder này với tên: **hero-bg.jpg** (hoặc .png, .webp)

## Yêu cầu
- Tên file: `hero-bg.jpg` (hoặc `hero-bg.png`, `hero-bg.webp`)
- Kích thước khuyến nghị: 1920x1080px hoặc lớn hơn
- Định dạng: JPG, PNG, hoặc WebP

## Vị trí
Ảnh này sẽ được sử dụng làm background cho hero section (phần đầu trang)
