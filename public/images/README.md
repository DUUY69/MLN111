# Thư mục Images

## Hướng dẫn thêm ảnh

### Hero Background Image
- **Tên file:** `hero-crowd.jpg` (hoặc `.png`, `.webp`)
- **Vị trí:** Đặt file ảnh vào thư mục này
- **Kích thước khuyến nghị:** 1920x1080px hoặc lớn hơn
- **Định dạng:** JPG, PNG, hoặc WebP

### Các ảnh khác
Bạn có thể thêm các ảnh khác vào đây và cập nhật đường dẫn trong CSS:
- `character.jpg` - Ảnh nhân vật
- `section-bg.jpg` - Background cho các section
- `gallery-1.jpg`, `gallery-2.jpg`, etc. - Ảnh gallery

## Lưu ý
- Đảm bảo ảnh có kích thước phù hợp để tối ưu hiệu suất
- Nên sử dụng định dạng WebP cho chất lượng tốt và kích thước nhỏ
- Ảnh sẽ tự động được Next.js tối ưu hóa
