# Feature Items Images

## Cách sử dụng
Đặt ảnh minh họa cho các feature items vào folder này:

## Tên file cần có:
- **feature-2-1.jpg** - Ảnh cho "Lực lượng sản xuất cơ bản" (2.1)
- **feature-2-2.jpg** - Ảnh cho "Động lực cách mạng" (2.2)
- **feature-2-3.jpg** - Ảnh cho "Sáng tạo văn hóa" (2.3)

## Yêu cầu
- Kích thước khuyến nghị: 400x300px
- Định dạng: JPG, PNG, hoặc WebP
- Ảnh sẽ hiển thị như background với opacity 0.2

## Vị trí
Ảnh này sẽ được sử dụng trong các feature boxes trong section "Vai trò quyết định của quần chúng nhân dân"
